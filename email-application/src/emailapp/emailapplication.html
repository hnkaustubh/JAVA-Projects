Email Application


Scenario: You are an IT Support Administrator and are charged with the task of creating Email IDs for New Hires.


Your Application should do the following -
1. Generate an Email with the following syntax - firstname.lastname@department.company.com
2. Determine the Department(sales,department,accounting), if none, leave blank
3. Generate a random String for password
4. Have Set Methods to change password , set the mailbox capacity and and define an alternate email address 5. Have Get Methods to display the name, email and mailbox capacity
